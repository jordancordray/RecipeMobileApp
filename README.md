# Meal Planner Mobile App

This app allows you to plan a meal for every day of the week, create new recipes to add to your list 
and shows a shopping list of ingredients for the meals you planned. 


## Instructions for Build and Use

Steps to build and/or run the software:

1. Install android studio
2. Clone this repository
3. Connect to the emulator or your own device
4. Build and run the software

Instructions for using the software:

1. Navigate to different screens (Home, Recipes, Shopping list) using the bottom navigation bar
2. Plan meals by clicking on the days on the home screen
3. Create new recipes in the new recipe tab of the recipes page
4. Select a recipe to show its details

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Android Studio (latest version)
* Kotlin version 2.0.21
